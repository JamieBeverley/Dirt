# dependencies #
* [liblo](http://liblo.sourceforge.net/)
  * FreeBSD: ```pkg install liblo```
  * Debian: ```apt-get install liblo-dev```
* [jack](http://jackaudio.org/)
  * FreeBSD: ```pkg install jackit```
  * Debian: ```apt-get install libjack-jackd2-dev jackd```
